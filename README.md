# Ohjelmoinnin Näyttö - Kellari Clicking Peli

Kellari clicking peli on tehty Ohjelmointi Näyttöä varten

### Muuta

Musiikki: Musiikit on kaikki hankittu Pixabay.com sivustosta, kaikki ilmaisia emmekä vaadi niitä omiksemme

Klikattavuus: Jos vaikuttaa että jotain esinettä esim todella pienikokoista ei pysty valitsemaan, klikkaa sen läheltä ja ulkopuolelta
(Vähäisen ajan, väsymyksen takia ei tarkenneta tiettyjen esineiden napsautettavuutta, on myös mahdollista että moinen ongelma on vain tietyillä käyttäjillä jos esim firefox käytössä)

Kuvat ja taustat: Kuvat ja taustat ovat tämän pelin tekijöiden omaa sisältöä
